## 🔍 체크 항목 상세

각 페이지마다 다음 항목을 상세하게 확인합니다:

1. **필터 라벨**: 모든 select, input에 명시적인 label 태그 존재 여부
2. **드롭다운**: CompanySelect 등 공통 컴포넌트 사용 여부, 옵션 로드 확인
3. **버튼 동작**: 모든 버튼의 onClick 핸들러 연결 상태
4. **API 연결**: 각 기능별 API 엔드포인트 확인
5. **기능 검증**: 실제 동작 여부 확인
