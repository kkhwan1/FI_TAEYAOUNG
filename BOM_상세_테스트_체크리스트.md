## 2. 필터 기능 테스트

### 2.1 납품처 필터링 (이슈 1-2)
- [ ] 납품처 드롭다운 클릭
- [ ] 납품처 목록 확인 (5개: 대우당진, 대우포승, 인알파코리아, 풍기서산, 호원오토)

- [ ] "대우포승" 선택
- [ ] 필터링된 BOM 데이터 로딩 확인
- [ ] 로딩 시간 측정 (1초 이내 권장)
- [ ] 필터링된 데이터만 표시 확인

### 2.2 공급처 필터링

- [ ] 공급처 목록 표시 확인
- [ ] 공급처 선택 시 필터링 작동 확인
- [ ] 납품처와 공급처 동시 필터링 확인

### 2.3 모품목 필터링

- [ ] 모품목 목록 표시 확인
- [ ] 모품목 선택 시 해당 모품목의 BOM만 표시 확인

### 2.4 차종 필터링
- [ ] 차종 필터 기능 확인
- [ ] 차종 선택 시 필터링 작동 확인

## 3. BOM 등록 기능 테스트

### 3.1 BOM 등록 다이얼로그 열기
- [ ] "BOM 등록" 버튼 클릭
- [ ] BOM 등록 다이얼로그 열림 확인
- [ ] 폼 필드 구성 확인:
  - 모품목 (필수)
  - 자품목 (필수)
  - 납품처 (선택)
  - 공급처 (선택)
  - 레벨 (기본값: 1)
  - 소요량 (필수)
  - 비고

### 3.2 납품처 드롭다운 확인 (BOM 등록 폼)
- [ ] 납품처 드롭다운 클릭
- [ ] 5개 납품처 목록 표시 확인
  - 대우당진
  - 대우포승
  - 인알파코리아
  - 풍기서산
  - 호원오토
- [ ] 각 항목에 "납품처" 라벨 표시 확인
- [ ] 공급처 항목이 섞여있지 않은지 확인

### 3.3 모품목 선택
- [ ] 모품목 검색 필드 클릭
- [ ] 검색 기능 작동 확인
- [ ] 모품목 선택
- [ ] 모품목 상세 정보 표시 확인

### 3.4 자품목 선택
- [ ] "코일 자재만 표시" 체크박스 기능 확인
- [ ] 자품목 검색 필드 클릭
- [ ] 검색 기능 작동 확인
- [ ] 자품목 선택
- [ ] 자품목 상세 정보 표시 확인

### 3.5 BOM 등록 저장
- [ ] 필수 필드 입력 (모품목, 자품목, 소요량)
- [ ] 납품처 선택 (예: 대우포승)
- [ ] 공급처 선택 (선택사항)
- [ ] "저장" 버튼 클릭
- [ ] 저장 성공 메시지 확인
- [ ] 다이얼로그 닫힘 확인
- [ ] 등록된 BOM이 목록에 추가되었는지 확인
