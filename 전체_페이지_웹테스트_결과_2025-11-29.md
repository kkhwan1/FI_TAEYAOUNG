### ✅ 2. 품목 관리 (`/master/items`)

**결과**: **정상**

- 페이지 로드 성공
- "품목 관리" 제목 확인
- 주요 기능 확인:
  - 검색 기능 (품목코드, 품목명, 규격, 소재로 검색)
  - 필터링 옵션:
    - 분류 필터 (원자재, 부자재, 반제품, 제품, 상품)
    - 타입 필터 (RAW, SUB, FINISHED)
    - 소재 필터 (COIL, SHEET, OTHER)
    - 차종 필터 (CK, GL3, TAM BEV 등 다수)
    - 거래처 필터
    - 도장상태 필터
  - 테이블 정상 표시 (총 276개 항목, 20개씩 페이징)
  - 작업 버튼 (이미지 관리, 품목 수정, 품목 삭제) 확인
  - 품목 내보내기 (20개) 버튼 확인

### ✅ 3. 생산 관리 (`/inventory?tab=production`)

**결과**: **기본 기능 정상, 일부 UI 이슈 확인**

- 페이지 로드 성공
- "생산 관리" 제목 확인
- 주요 기능 확인:
  - 실시간 재고 현황 섹션 표시
  - 최근 거래 내역 테이블 표시 (2건)
  - 생산 등록 버튼 정상 작동
  - 생산 등록 모달 정상 오픈
  - 공정 구분 체크박스 (프레스, 용접, 도장) 표시
  - 프레스 용량 드롭다운 정상 오픈
    - 옵션: 400톤, 600톤, 800톤, 1000톤, 1600톤
  - 고객사 (주문처) 드롭다운 표시
  - BOM 자동 차감 체크박스 표시

**이슈 확인:**
- ⚠️ **프레스 용량 선택 시 UI 레이어 문제**: 드롭다운은 정상 오픈되나, 옵션 선택 시 타임아웃 발생 (레이어/z-index 이슈 추정)
- ⚠️ **API 500 에러**: 고객사 및 재고 정보 조회 시 500 에러 발생 (에러 메시지: "Error fetching companies", "Failed to fetch stock info")

**테스트 항목:**
- ✅ 이슈 2-1 (프레스 용량 선택): 드롭다운 정상 오픈 확인 (선택 기능은 UI 레이어 문제로 추가 확인 필요)
- ⏸️ 이슈 2-2 (용접 공정 고객사 필터링): API 에러로 인해 완전한 테스트 불가
- ⏸️ 이슈 2-2-1 (공정별 모품목 리스트 분리): 프레스 용량 선택 미완료로 테스트 불가
- ⏸️ 이슈 2-3 (코일 자동 차감): 백엔드 로직 확인 필요 (UI 레벨에서는 체크박스 표시 확인)
