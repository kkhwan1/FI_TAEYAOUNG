### ✅ 1. 이슈 2-2: 용접 공정 고객사 선택 표시 라벨

**확인 방법**:
1. 브라우저에서 `http://localhost:5000/production` 접속
2. "생산 등록" 버튼 클릭
3. "용접" 체크박스 선택
4. "고객사" 드롭다운 클릭
5. 드롭다운 목록 확인

**확인 사항**:
- 각 회사 옆에 표시된 라벨 확인
  - ✅ 정상: "납품처" 표시
  - ❌ 문제: "공급업체" 표시

**확인 대상 회사**:
- 대우당진
- 대우포승
- 인알파코리아
- 풍기서산
- 호원오토

**코드 분석 결과**:
- `src/components/CompanySelect.tsx` 292줄에서 `companyType === 'CUSTOMER'`일 때 '납품처'로 표시하도록 구현됨
- `src/components/production/ProductionEntryForm.tsx` 627줄에서 `companyType="CUSTOMER"` 설정

**예상 결과**: ✅ 정상 작동 (코드상 수정 완료)

### ⚠️ 3. 이슈 1-2: BOM 납품처 필터링 오류

**확인 방법**:
1. 브라우저에서 `http://localhost:5000/master/bom` 접속
2. "납품처" 드롭다운에서 "대우포승" 선택
3. BOM 테이블 확인
4. "BOM 등록" 또는 "수정" 버튼 클릭
5. 등록/수정 다이얼로그에서 "납품처" 드롭다운 확인

**확인 사항**:
- 드롭다운에 표시되는 납품처 목록
  - ✅ 정상: "인알파코리아", "풍기서산" 등 실제 납품처 표시
  - ❌ 문제: "대우포승"만 표시되거나 공란

**추가 확인**:
- BOM 테이블의 "납품처" 컬럼에 다양한 납품처가 표시되는지 확인

**예상 결과**: ⚠️ 추가 확인 필요 (코드 분석 필요)

### ⚠️ 4. 이슈 2-2-1: 공정별 모품목 리스트 분리

**확인 방법**:
1. 브라우저에서 `http://localhost:5000/production` 접속
2. "생산 등록" 버튼 클릭
3. "프레스" 체크박스 선택
4. "용량" 드롭다운에서 용량 선택:
   - 테스트 1: 블랭킹용 (400톤 또는 600톤) 선택
   - 테스트 2: 성형용 (1000톤 또는 1600톤) 선택
5. 각 용량 선택 후 "모품목" 또는 "품목" 드롭다운 확인

**확인 사항**:
- 블랭킹(400~600톤) 선택 시: 블랭킹용 모품목만 표시되는지
- 성형(1000~1600톤) 선택 시: 성형용 모품목만 표시되는지
  - ✅ 정상: 용량에 따라 다른 모품목 리스트 표시
  - ❌ 문제: 용량과 관계없이 동일한 모품목 리스트 표시

**코드 분석 결과**:
- `src/components/production/ProductionEntryForm.tsx`에서 프레스 용량 선택 기능은 구현됨
- 하지만 용량에 따라 모품목을 필터링하는 로직은 확인되지 않음

**예상 결과**: ❌ 문제 가능성 (블랭킹/성형 구분 로직 확인 필요)
