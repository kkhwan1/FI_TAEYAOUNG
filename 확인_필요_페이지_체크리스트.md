## 🔍 체크 항목 상세 설명

### 필터 라벨 확인
- 모든 select, input 요소에 명시적인 `<label>` 태그가 있는지 확인
- placeholder만 있는 경우 라벨 추가 필요

### 드롭다운 확인
- CompanySelect 컴포넌트 사용 여부
- 옵션이 제대로 로드되는지 확인
- 선택 시 동작 확인

### 버튼 동작 확인
- 등록/수정/삭제 버튼
- 엑셀 업로드/다운로드 버튼
- 인쇄 버튼
- 필터 초기화 버튼
- 새로고침 버튼

### API 연결 확인
- 각 버튼 클릭 시 올바른 API 호출
- 필터 변경 시 API 파라미터 전달 확인
- 에러 처리 확인
